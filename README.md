# Criando um Podcast com I.A.'s Generativas

Esse conteúdo foi desenolvido para conhecimento e uso de I.A Generativa, explorando a criação de imagens e texto para voz através de prompt's, com a finalidade de desenvolver um podcast sobre Java e Inteligência artificial.

> Não foi desenvolvido e nem utilizada nenhuma linguagem de programação, apenas as ferramentas existentes no desenvolvimento de conteúdo para absorção de ideias para projetos futuros.


## Uso de LLM's

Feito o uso dessas ferramentas para o prompt, identificando o padrão de cada uma delas e as saídas diferentes para o mesmo contexto.

- [Copilot](https://copilot.microsoft.com)
- [Deepseek](https://chat.deepseek.com)
- [Gemini](https://gemini.google.com/app)
- [ChatGpt](https://chatgpt.com/)

1. Copilot possui liberadade para desenvolvimento de imagens, diferente do Leonardo.AI que é pago e tem limite para uso, conseguir gerar apenas duas imagens.
2. Deepseek muito completo na desenvoltura do contexto, o retorno foi satisfatório.
3. Configurei o Gemini com uma persona específica, engenheiro de prompt com experiência de 20 anos, retorno também satisfatório.
4. O Chatgpt não usei tanto, devido a limitação, porém as respostas que foram dadas foram satisfatórias.

## Ferramentas de desenvolvimento a partir do prompt

- [Leonardo.Ai](https://leonardo.ai/) Possível gerar 2 imagens gratuitamente. 
- [Copilot](https://copilot.microsoft.com/) Dos chats existentes, com o Copiot da Microsoft eu consegui fazer várias alterações da imagem, sem limite e chegando ao resultado esperado. A renderização da imagem ficou muito boa e recomendo explorar o Copilot web.
> As ferramentas acima são em substituição da midjourney que é paga.


- [Luvvoice](https://luvvoice.com/) - Ferramenta gratuita como alternativa ao Eleven Labs (pago).

- [Capcut](https://www.capcut.com/activities/subscribe/?irclickid=y-9xSNWT1xycRzIXYITn3WaoUkpU4SW-Zx51RE0&sharedid=&irpid=1943907&irgwc=1&afsrc=1) Edição de vídeo gratuito, áudio e entre outros. Possui versão pro.
